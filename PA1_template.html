Examining the plot reveals that we have a lot of data points that are close to zero measurements. This may suggest that a person shifts his activity during the day. For the data analysis we might prefer to study cumulative activity per day.

Missing values for the 5-minute time period are more likely to be simply zeroes with this hypothesis; if we are missing values for the day entirely we might want to consider a different approach.
